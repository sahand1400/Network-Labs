HQ-1 IPv4 OSPF 
--------------
router ospf 1
network 192.168.1.1 0.0.0.0 area 0
network 3.3.3.1 0.0.0.0 area 0
network 1.1.1.1 0.0.0.0 area 1
passive-interface fa0/0

inter fa 1/1
ip ospf hello-interval 60
ip ospf dead-interval 180

inter s 2/1
ip ospf hello-interval 60
ip ospf dead-interval 180
