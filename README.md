## 📚 About This Repository

This collection contains **hands-on labs and tested scenarios** for **CCNP Enterprise** certification including **ENCOR (350-401)** and **ENARSI (300-410)**.

All scenarios are based on **Hamed Banifatemeh's** teaching methodology and other trusted resources. Each lab includes:

- ✅ **Topology diagram** (router names, interfaces, IP addresses, OSPF areas)
- ✅ **Initial and final configuration files**
- ✅ **Step-by-step scenario and solution**
- ✅ **Verification commands with output analysis**
- ✅ **Troubleshooting scenarios**

**Note:** ENCOR and ENARSI topics are integrated within each lab scenario rather than separated into different folders. Each lab covers both core concepts and troubleshooting skills together.
